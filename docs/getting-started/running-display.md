## Before you start

You'll need a few things in place first:

- An active seat on the DSP you're activating in
- A Chalice deal ID (request one from your Chalice contact)
- Creative assets ready to upload, or a third-party tag

!!! note
    Deals usually take a few hours to sync from the DSP to the exchange. If you don't see avails right away, give it a little time before raising a flag.

## Step-by-step setup

1. Request a display deal ID from your Chalice contact.
2. Accept the deal in your platform's deals or inventory section.
3. Create an insertion order and a display line item.
4. Attach the deal to the line item and set your budget and flight dates.
5. Save and set the line item live.

!!! tip
    Keep targeting broad to start. Layering on too many filters early can choke delivery before the model has data to optimize against.

## Two ways to add creative

=== "Option 1: Third-party tag"

    1. Select **New creative**, then **Third-party**.
    2. Choose **Third-party display**.
    3. Paste your JavaScript tag, set the correct size, and add the landing page URL.
    4. Save.

=== "Option 2: Hosted banner"

    1. Select **Upload**, then **Image or HTML5**.
    2. Add your creative file (JPG, PNG, or HTML5) and the landing page URL.
    3. Save.

## Optional creative enhancements

Once your creatives are live, you can layer on Chalice creative enhancements at no additional cost.

| How to apply | What happens |
| --- | --- |
| Send your creative IDs to the Chalice team and note which enhancement you'd like for each. | We handle the technical setup. Once the creatives run via the deal, the enhancements activate automatically. |

!!! warning
    Double check the creative IDs before sending. Sending the wrong IDs is the most common reason enhancements don't show up on the right placements.
